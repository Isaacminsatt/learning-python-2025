# Learning Python 2025

A Python program that provides course platform recommendations based on student preferences.

## What it does
- Asks for grade level, favorite subject, and available study time
- Provides text-based recommendations for online learning platforms
- Suggests different platforms based on the user's criteria

## How to run
```python
python Course_Recommendation.py
```

## Features
- Interactive user input
- Conditional logic based on grade, subject, and time preferences
- Provides platform suggestions (like Khan Academy, Coursera, etc.) as text output
